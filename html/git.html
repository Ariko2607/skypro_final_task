Git

  • git init - Инициализация нового репозитория.
  • git clone -Клонирование существующего репозитория.
  • git config - Настройка параметров Git.На пример git config --global user.name "Your Name".
  • git status - Показать состояние рабочей директории.
  • git log - Показать историю коммитов.
  • git add . - Добавить все измененные файлы.
  • git add -A - Добавить все файлы (включая удаленные)
  • git commit -am - Добавить и закоммитить все изменения.
  • git branch - Показать список веток.
  • git branch name - Создать новую ветку.
  • git checkout - Переключиться на ветку.
  • git merge - Объединить ветки.
  • git push - Отправить изменения на сервер.
  • git pull - Получить изменения сервера
  • git reset - Отменить коммит но сохранить изменения
  • git reset --hard - Отменить коммит но удалить изменения
